                              DEFINTIONS OF AWS SERVICES

Analytics :  = 9
 
   1)Amazon Athena:
Amazon Athena is a serverless, query-based service that allows you to analyze data stored in Amazon S3 using standard SQL queries, without the need for managing infrastructure.

2)AWS Data Exchange:
AWS Data Exchange is a service that simplifies the discovery, subscription, and use of third-party data, providing a secure platform for data exchange in the cloud.

3)Amazon EMR:
Amazon EMR (Elastic MapReduce) is a cloud-based big data platform that facilitates the processing of large amounts of data quickly and cost-effectively, using popular frameworks like Apache Spark and Hadoop.

4)AWS Glue:
AWS Glue is a fully managed extract, transform, and load (ETL) service that makes it easy to prepare and load data for analysis, simplifying the data integration process.

5)Amazon Kinesis:
Amazon Kinesis is a platform for building real-time streaming data applications, allowing you to ingest, process, and analyze streaming data at scale.

6)Amazon MSK (Managed Streaming for Apache Kafka):
Amazon MSK is a managed service that simplifies the deployment, operation, and scaling of Apache Kafka, a popular open-source streaming platform.

7)Amazon OpenSearch Service:
Amazon OpenSearch Service is a fully managed service for deploying, operating, and scaling Elasticsearch, an open-source search and analytics engine.

8)Amazon QuickSight:
Amazon QuickSight is a business analytics service that enables users to create visualizations and perform ad-hoc analysis of their data.

9)Amazon Redshift:
Amazon Redshift is a fully managed data warehouse service that allows you to run complex queries and perform analytics on large datasets with high performance and scalability.

Application Integration : = 4 

10)Amazon EventBridge:
Amazon EventBridge is a serverless event bus service that makes it easy to connect and route events between different applications using a simple, scalable, and event-driven model.

11)Amazon Simple Notification Service (Amazon SNS):
Amazon SNS is a fully managed messaging service that enables the sending of messages or notifications to a distributed set of recipients, such as end users or other distributed services.

12)Amazon Simple Queue Service (Amazon SQS):
Amazon SQS is a fully managed message queuing service that enables the decoupling of components in a distributed system by allowing the transmission of any volume of data, at any level of throughput, without losing messages or requiring other services to be available.

13)AWS Step Functions:
AWS Step Functions is a serverless function orchestrator that allows users to coordinate and sequence AWS services and functions in workflows, making it easier to build and maintain distributed applications.

Business Applications: = 2

14)Amazon Connect:
Amazon Connect is a cloud-based contact center service that enables organizations to set up and manage a customer contact center quickly and efficiently, facilitating seamless customer interactions across various communication channels.

15)Amazon Simple Email Service (Amazon SES):
Amazon SES is a scalable and cost-effective email service that allows businesses to send and receive emails securely, using a reliable and highly available infrastructure. It is designed to simplify the process of sending transactional, marketing, and notification emails.

Cloud Financial Management: = 5

16)AWS Billing Conductor:
AWS Billing Conductor is a service that aids in creating and managing billing agreements, providing a toolset for organizing and controlling financial aspects within the AWS environment.

17)AWS Budgets:
AWS Budgets is a feature that enables users to set custom cost and usage budgets, allowing for proactive monitoring and alerts when expenditures exceed predefined thresholds.

18)AWS Cost and Usage Report:
AWS Cost and Usage Report provides comprehensive data about costs and usage, offering detailed insights to help users analyze and manage their AWS spending.

19)AWS Cost Explorer:
AWS Cost Explorer is a tool that allows users to visualize, understand, and manage AWS costs and usage over time, providing insights to optimize resource utilization and control expenses.

20)AWS Marketplace:
AWS Marketplace is an online store that simplifies the discovery, purchase, and deployment of a wide range of software, services, and data directly onto AWS infrastructure, providing a centralized platform for vendors and customers to interact.

Compute: = 7

21)AWS Batch:
AWS Batch is a fully managed service that enables users to run batch computing workloads efficiently by dynamically provisioning the optimal quantity and type of compute resources.

22)Amazon EC2:
Amazon Elastic Compute Cloud (Amazon EC2) provides resizable compute capacity in the cloud, allowing users to run virtual servers, known as instances, for various applications.

23)AWS Elastic Beanstalk:
AWS Elastic Beanstalk is a fully managed service that simplifies the deployment and scaling of web applications, handling the infrastructure details while users focus on application development.

24)Amazon Lightsail:
Amazon Lightsail is a simplified compute platform that offers easy-to-use cloud services, enabling users to deploy, manage, and scale applications and websites.

25)AWS Local Zones:
AWS Local Zones are extension locations that bring AWS services closer to end-users and specific geographic areas, providing low-latency access to compute and storage resources.

26)AWS Outposts:
AWS Outposts is a fully managed service that extends AWS infrastructure, allowing users to run AWS services locally on-premises for specific workloads.

27)AWS Wavelength:
AWS Wavelength is a service that brings AWS services to the edge of 5G networks, enabling ultra-low latency applications for use cases like augmented reality and gaming.

Containers: =3

28)Amazon Elastic Container Registry (Amazon ECR):**
Amazon Elastic Container Registry (Amazon ECR) is a fully managed Docker container registry that makes it easy for developers to store, manage, and deploy Docker container images, seamlessly integrating with other AWS services and providing secure and scalable storage for container artifacts.

29)Amazon Elastic Container Service (Amazon ECS):**
Amazon Elastic Container Service (Amazon ECS) is a fully managed container orchestration service that simplifies the deployment, management, and scaling of containerized applications using Docker containers, allowing users to focus on building applications rather than managing the underlying infrastructure.

30)Amazon Elastic Kubernetes Service (Amazon EKS):**
Amazon Elastic Kubernetes Service (Amazon EKS) is a fully managed Kubernetes service that makes it easy to deploy, manage, and scale containerized applications using Kubernetes, simplifying the process of running Kubernetes on AWS infrastructure and ensuring high availability and security for containerized workloads.

Customer Engagement: = 4

31)AWS Activate for Startups:**
AWS Activate for Startups is a program that provides eligible startups with credits, technical support, training, and other resources to help them build and scale their businesses on the AWS cloud.

32)AWS IQ:**
AWS IQ is a service connecting AWS customers with AWS Certified freelancers and consulting firms, facilitating on-demand expertise for various cloud-related projects.

33)AWS Managed Services (AMS):**
AWS Managed Services (AMS) is a fully managed service that helps enterprises migrate to the cloud and operate their infrastructure securely and efficiently, with AWS managing routine maintenance tasks and providing ongoing support.

34)AWS Support:**
AWS Support is a subscription service that offers access to a range of resources, including technical support plans, documentation, and forums, to help AWS customers effectively use and troubleshoot AWS services.

Database: = 5

35)Amazon Aurora:**
Amazon Aurora is a fully managed relational database service that is compatible with MySQL and PostgreSQL, offering high performance, scalability, and durability for database workloads.

36)Amazon DynamoDB:**
Amazon DynamoDB is a fully managed NoSQL database service that provides seamless and fast access to data, allowing users to build applications with flexible and scalable data models.

37)Amazon MemoryDB for Redis:**
Amazon MemoryDB for Redis is a fully managed, Redis-compatible, in-memory database service that enables users to build high-performance, scalable applications with sub-millisecond latency.

38)Amazon Neptune:**
Amazon Neptune is a fully managed graph database service that allows users to build and run applications that work with highly connected datasets, such as social networks and recommendation engines.

39)Amazon RDS:**
Amazon Relational Database Service (Amazon RDS) is a fully managed relational database service that supports multiple database engines, including MySQL, PostgreSQL, Oracle, SQL Server, and MariaDB, simplifying database administration tasks.
